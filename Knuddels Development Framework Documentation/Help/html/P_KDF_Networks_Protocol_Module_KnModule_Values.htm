public abstract class ProductBean
extends java.lang.Object
implements PropertiesExtendible, Log, BuildFilterable

The ProductBean abstract class is the base class for the elements that compose the installation ProductTree. This class provides the basic functionality required by all ProductBeans (e.g., ProductFeatures, ProductComponents, ProductActions, etc.) to allow proper run-time functionality through the ProductService. In addition, the ProductBean base class provides methods that allow access to the core functionality of an installation (e.g., services, logging capabilities, string resolver, etc.).

CAUTION : This class should not be extended directly when creating a custom ProductBean. Custom ProductBeans should extend the ProductAction class.

NOTE : The ProductBean class supports the PropertiesExtendible interface. Supporting that interface allows the members of the class to be extended by a PPK. This allows a PPK to add the required additional information and parameters to certain objects for use on the platform(s) supported by the PPK. For example, the Linux PPK utilizes this functionality to allow Linux platform-specific interaction between the "ProductComponent" class and the Linux RPM.
