Logs the specified event for post-install display purposes. This method can be used by ProductActions to allow them to provide informational messages (eg. warnings, errors, etc.) that are displayed through the post-install summary panel.

NOTE : The Log interface provides typical message types that should be used when logging events. For example, the following code could be used to log a warning:

support.logEvent(this, Log.WARNING, "This is a warning.");

The following code could be used to log an error:

support.logEvent(this, Log.ERROR, "This is a error!!!");
